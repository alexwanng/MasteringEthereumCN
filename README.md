# Mastering Ethereum 精通以太坊中文版翻译


## 引子
在学习以太坊过程中， 最后觉得以太坊白皮书、黄皮书、以及这本精通以太坊比较经典， 黄皮书公式概念比较集中， 这本精通以太坊则是循序渐进， 有朋友说还没有中文版， 便有意组织一下翻译， 大家可以选择自己想翻译的章节。
